# catalogoMYLVA

Aplicación web para seleccionar productos del catálogo MYLVA según:

- especie de plaga
- entorno de uso
- extensión del tratamiento

La aplicación consulta la base de datos `productos.json` y devuelve:

- producto recomendado
- registro sanitario
- forma de uso
- dosis

## Uso

Abrir la página web e introducir:

1. especie de plaga
2. entorno
3. superficie

La aplicación mostrará el producto correspondiente del catálogo.
